# paynch-php-examples
PHP integration examples for Paynch – non-custodial crypto payments on EVM chains (BSC, Polygon, Arbitrum, Base).

Includes:
- Secure backend validation (PDO lock, random orderId, 1.2% tolerance)
- API verification (/paynch.php)
- Payment button embed + polling

Accept USDT or any custom token. No KYC. Always validate server-side!
